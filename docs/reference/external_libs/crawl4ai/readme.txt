Crawl4AI external reference

Purpose
- Durable reference for Crawl4AI usage patterns and extracted upstream context.
- Use this folder instead of the old single huge file.

Index
- llm_extraction.md - LLM extraction guidance and caveats
- non_llm_extraction.md - preferred non-LLM extraction strategies
- multi_url_crawling.md - batch and streaming multi-URL crawling
- http_crawler.md - HTTP crawler strategy and patterns
- url_seeder.md - URL seeding and discovery
- filters_and_scorers.md - deep crawl filters, scorers, and ranking
- configuration_objects.md - browser, crawler, and LLM configuration reference
- simple_crawling.md - simple crawling, table extraction, and common flows
- c4a_script_reference.md - C4AScript syntax for compiled fast-path execution

Agent reference (live docs)
