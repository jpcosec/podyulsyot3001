# Seed Documents

Foundation documents for the PhD 2.0 documentation system.

## Structure

```
docs/seed/
├── product/     # PhD 2.0 product knowledge
├── practices/   # Documentation methodology and templates
└── README.md
```

| `implementation_status.md` | **Tracks implemented vs. planned features** |

### `practices/` — Documentation Methodology

Meta-documentation about HOW to work with the system:

| File | Description |

- **Product docs** → Answer "What does PhD 2.0 do?"
