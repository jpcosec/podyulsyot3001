### 📝 Abstract
This paper presents the development and validation of an Intelligent Tutoring System (ITS) for mathematics (Systems of Linear Equations). The system architecture integrates affective computing to monitor student stress and cognitive states via facial expression analysis. By computing a real-time stress index, the tutor provides adaptive, affective feedback to support autonomous learning. Functional validation with experts has been completed, identifying key UX and detection redundancies that informed the system's current iteration.

### 🛠️ Key Contributions
- **Affective Computing Pipeline:** Developed a computer vision-based emotional input processor for real-time stress index calculation.
- **Adaptive Feedback:** Engineered a pedagogical engine that triggers interventions based on the student's affective state during 3x2 SLE tasks.
- **Expert Validation:** Led the functional testing and iterative redesign of the tutor's interface and detection modules.
